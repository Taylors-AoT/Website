## Run Locally

**Prerequisites:**  Node.js (version 20.x.x)
**If not installed/different version** https://github.com/coreybutler/nvm-windows/releases


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
